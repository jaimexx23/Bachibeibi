# BachilleresQR

Proyecto migrado a Laravel que contiene la aplicación de registro y control de asistencias.

Instrucciones rápidas para subir a GitHub:

1. Crea un repositorio vacío en tu cuenta llamado `edsoncorazon` (SSH).
2. Desde esta carpeta ejecuta:

```bash
git remote add origin git@github.com:edsoncorazon/edsoncorazon.git
git push -u origin main
```

Si prefieres que yo empuje, crea el repo en GitHub y dime cuando esté listo.
